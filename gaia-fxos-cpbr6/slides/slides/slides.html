Open Web Apps

Web apps são aplicativos construídos usando tecnologia padrão. Eles funcionam em qualquer navegador moderno.


O projeto de Open Web Apps propõe algumas pequenas adições para sites existentes para transformá-los em apps.


Estes aplicativos podem ser executados em navegadores de desktop e dispositivos móveis.

Começando a construir aplicativos - https://developer.mozilla.org/pt-BR/docs/Apps/Getting_Started

  1. Desenvolva Web App usando HTML5, CSS, e Javascript
  2. Crie um arquivo app manifest
  3. Publique/Instale o app

Obtendo um exemplo de manifest.webapp

curl -I http://mozillalabs.com/manifest.webapp

Níveis de segurança

Concedido por padrão

As WebAPIs seguras que não expõem a privacidade dos dados importantes. WebGL, tela cheia, áudio, etc.

Concedido pelo usuário

Localização, câmera, acesso ao sistema de arquivos

Concedido quando instalado

LocalStorage, IndexedDB, oﬄine cache

Concedido pela loja autorizada

APIs de privacidade e segurança, como API de contatos

Verificado por assinatura

APIs altamente privilegiadas, como acesso por rádio (discador)
